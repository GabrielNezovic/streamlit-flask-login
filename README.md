# streamlit-flask-login
 A barebones template for encrypted User Authentication using Streamlit as a front-end web UI and Flask as a back-end API server.
 
___

Start both servers at the same time with:
```
run.bat
```

___

Login Credentials:

```
user1: password1
user2: password2
```
